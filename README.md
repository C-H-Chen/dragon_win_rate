# 簡介:

針對預測球隊年度勝率之問題，嘗試藉此掌握影響勝率之關鍵因子，洞察各項因子隨著數值變化，會如何影響勝率，達到設定球員篩選標準之目的。

# 數據清洗:  

發現"球員打擊(2010-24)"的銀棒指數第862行與"球員守備(2010-24)"的守備率欄位第2907行有缺失值，以填入實際正確數值做處理。  

發現1998味全龍多個球隊相關數值出現異常值，但本研究決定以2010-24年的數據進行研究，故不影響。
